Question 2

2017 P1 Q3
For ⁴⁰₁₉K: protons = 19, electrons = 19, neutrons = 40 − 19 = 21
Select the answer from your copy of the paper:
💡 Explanation
The proton number (bottom number) of potassium is 19, so it has 19 protons. As a neutral atom, it also has 19 electrons. The nucleon number (top number) is 40, which equals protons + neutrons, so neutrons = 40 − 19 = 21.

Question 5

2020 P1 Q4
Isotopes: same proton number, different neutron number; same electron arrangement
Select the answer from your copy of the paper:
💡 Explanation
Isotopes of the same element always have the same proton number — that is what makes them the same element. As neutral atoms, they also have the same number of electrons and therefore the same electron arrangement and the same number of electron shells. What differs between isotopes is the number of neutrons (and hence the nucleon number).

Question 6

2024 P1 Q4
In ᵃᵦX: b = proton number (bottom), a = nucleon number (top); neutrons = a − b
Select the answer from your copy of the paper:
💡 Explanation
In nuclide notation ᵃᵦX, b is the proton number (bottom) and a is the nucleon number (top). The position of X in the Periodic Table is determined by its proton number b. A neutral atom of X has b electrons (not a − b). The number of neutrons = a − b, but the proton count itself is b.
⚡ One-line Takeaway

In nuclide notation, the bottom number is the proton number, the top number is the nucleon number, and neutrons = nucleon number − proton number; isotopes share the same proton number but differ in neutron number.
